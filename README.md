# ⏱️ Contador Digital

Um projeto de contador digital que exibe o tempo em blocos de horas, minutos e segundos, com tema escuro e interface limpa.

Ideal para timers, contagens regressivas e relógios de tela.

## ✨ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 🎨 Características do Design

- Visual minimalista e moderno
- Paleta de cores escura e contrastante
- Blocos com efeito de profundidade
- Tipografia ampla e de fácil leitura
- Design responsivo

## 🚀 Objetivo do Projeto

Praticar a construção de componentes dinâmicos de interface, explorando manipulação de tempo com JavaScript, estilização com CSS e atualização em tempo real dos elementos na tela.

## 📚 Funcionalidades

- Exibição de horas, minutos e segundos em blocos separados
- Atualização automática em tempo real
- Estrutura adaptável para contagem regressiva ou relógio
- Formatação de números com dois dígitos (00, 45, 17)
- Interface leve e responsiva

## 💼 Possíveis Aplicações

- Contagem regressiva para eventos
- Relógio digital de tela
- Timer para tarefas e estudos
- Componente para landing pages
